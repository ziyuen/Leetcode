## 使用 stringstream 和 getline 分割字符串
```cpp
int main() {
	string str;	
	string str_cin("one#two#three");
	stringstream ss;
	ss << str_cin;

	while (getline(ss, str, '#'))
		cout << str<< endl;

	return 0;
}
```

output:
```
one
two
three
```
